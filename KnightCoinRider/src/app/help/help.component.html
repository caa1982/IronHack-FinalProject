Manual Command

how to trade crypto?
will return a list of videos and articles to learn about cryptocurrcies and trading

check "ticker or name"
will return the ticker, price in USD, price in BTC and daily % move of coin

coins "exchange"
will return the coins listed on the exchange

portfolio "exchange"
will return the amount of coins held

buy "ticker" @ "price" from "exchange"
will buy or add an order to the exchange

sell "ticker" @ "price" from "exchange"
will sell or add an order to the exchange

open orders "ticker" "exchange"
will return the open orders

trade history "ticker" "exchange"
will return the trades history

order book "ticker" "exchange"
will return the order book for a specific coin

wiki "word"
will search wikipedia for an article
